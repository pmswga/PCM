Документация по использованию PCM
Список всех функций, переменных, макроопределений, перечислений и определений типов со ссылками на файлы, к которым они принадлежат.
  • ABSTRACT_CLASS : ptypeclass.consts.php
  • ABSTRACT_METHOD : ptypemethod.consts.php
  • NATIVE_CLASS : ptypeclass.consts.php
  • NATIVE_METHOD : ptypemethod.consts.php
  • NATIVE_VAR : ptypevar.consts.php
  • PRIVATE_ACCESS : ptypeaccessright.consts.php
  • PROTECTED_ACCESS : ptypeaccessright.consts.php
  • PUBLIC_ACCESS : ptypeaccessright.consts.php
  • STATIC_METHOD : ptypemethod.consts.php
  • STATIC_VAR : ptypevar.consts.php
