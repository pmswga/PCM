Подробное описание

Менеджер для работы с пользователями

Автор
pmswga
Версия
1.0

Задачи менеджера:

  1. Добавлять/удалять пользователей
  2. Изменять информацию об пользователе
  3. Удалять пользователей, срок аккаунта которых истёк
